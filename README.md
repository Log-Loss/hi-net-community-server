# Hi, Net Community *server*
The backend of the Hi, Net Community
   
   
## Requirements
* JDK 1.8

## Getting Start
1. replace the mongoDB url in `application.properties`
2. run the Spring Boot Project
3. By default, it's on `localhost:8080`
